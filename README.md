
# 🧮 Python Calculator

A simple command-line calculator built using Python. This project performs basic arithmetic operations, stores calculation history, and handles invalid inputs using exception handling.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📜 View Calculation History
- ⚠️ Error Handling with `try` and `except`
- 🚪 Exit Option

## 🛠️ Technologies Used

- Python 3

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/python-calculator.git
```

2. Open the project folder.

3. Run the program:

```bash
python calculator.py
```
